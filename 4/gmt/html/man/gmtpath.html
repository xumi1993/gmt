NAME

gmtpath − Get full pathname for gmt files

SYNOPSIS

gmtpath leg-ids

DESCRIPTION

gmtpath returns the full pathname to one or more gmt-files. The pathname returned for a given cruise may change with time due to reshuffling of disks/subdirectories.
leg-ids

Can be one or more gmtleg-names, like c2610. Use ‘cat list_of_legs‘ to pass all the legnames in the file list_of_legs.

EXAMPLES

To obtain pathnames for cruises c2603 and v2819, run

gmtpath c2603 v2819
