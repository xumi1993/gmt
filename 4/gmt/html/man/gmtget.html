NAME

gmtget − To list a single GMT parameter

SYNOPSIS

gmtget PARAMETER

DESCRIPTION

gmtget will list the value of a single GMT default parameter.

EXAMPLE

To list the value of the parameter PS_VERBOSE:

gmtget PS_VERBOSE

SEE ALSO

GMT(1), gmtdefaults(1), gmtset(1)
